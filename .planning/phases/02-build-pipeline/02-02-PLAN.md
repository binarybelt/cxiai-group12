  <action>
1. Create `src/lib/convex-client.ts`:
   - Create a `ConvexHttpClient` instance using `process.env.NEXT_PUBLIC_CONVEX_URL`.
   - Export a `logGeneration(action: string, details: string)` helper that calls the `logAction` mutation with actor "build-pipeline".
   - Guard: if `NEXT_PUBLIC_CONVEX_URL` is not set, log a warning and skip (don't crash the pipeline for missing Convex).

2. Create `src/app/api/interpret-brief/route.ts`:
   - POST handler that extracts `{ brief }` from request body.
   - Validate brief is a non-empty string; return 400 if invalid.
   - Load system prompt from `src/agents/brief-interpreter/prompt.md` using `fs.readFile`.
   - Inject design system context into placeholders: `{{PATTERNS}}` -> `loadPatterns()`, `{{MARKETS}}` -> `loadMarkets()`, `{{COMPONENTS}}` -> `loadComponents().map(c => ({ id: c.id, category: c.category, description: c.description }))`.
   - Call `generateObject` with `anthropic("claude-sonnet-4-5-20250414")`, `BriefInterpretationSchema`, the system prompt, and the brief as user prompt. Set `maxRetries: 2`.
   - IMPORTANT: Verify the actual import path for anthropic provider during implementation. It should be `import { anthropic } from "@ai-sdk/anthropic"` — confirm this works after package install.
   - On success, call `logGeneration("interpret-brief", JSON.stringify({ brief: brief.slice(0, 200), interpretation: object }))`.
   - Return `Response.json(object)`.
   - Wrap in try/catch; return 500 with `{ error: "Brief interpretation failed" }` on failure.

3. Create `src/app/api/generate-page/route.ts`:
   - POST handler that extracts the `BriefInterpretation` from request body.
   - Load system prompt from `src/agents/component-selector/prompt.md` using `fs.readFile`.
   - Build the prompt by injecting: market config, required components for that market, the matching pattern, all components with their prop shapes (from `PROP_SHAPES`), and the available token IDs.
   - Call `buildConstrainedPageSpecSchema()` to get the constrained schema.
   - Call `streamObject` with `anthropic("claude-sonnet-4-5-20250414")`, the constrained schema, system prompt, and `JSON.stringify(interpretation)` as user prompt. Set `maxRetries: 2`.
   - Use `result.toTextStreamResponse()` to return the streaming response (NOT `toDataStreamResponse` — `useObject` expects text stream format per research).
   - IMPORTANT: Verify the `streamObject` + `toTextStreamResponse` API during implementation. The AI SDK v6 Output API may use slightly different patterns — check `import { streamObject } from "ai"` works correctly.
   - After stream completes, log to Convex via the `onFinish` callback if available, or fire-and-forget after returning the stream.
   - Wrap in try/catch; return 500 with `{ error: "Page generation failed" }` on failure.
  </action>
