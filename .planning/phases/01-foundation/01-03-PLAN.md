   - Is a React functional component with TypeScript props interface
   - Uses Tailwind classes referencing the Pfizer token extensions (e.g., `bg-pfizer-blue-500`, `text-heading-xl`)
   - Accepts the props defined in components.json (see interfaces above)
   - Has a default export AND a named export
   - Handles missing/optional props gracefully (sensible defaults)

   Component-specific requirements:

   **Hero** (`hero.tsx`): Full-width section with title (heading-xl), subtitle, optional background image, CTA button. Variants: fullWidth (100vw), split (image left, text right), centered (text centered). Default variant: fullWidth.

   **Card** (`card.tsx`): Rounded container with optional image, title, body text, CTA link. Variants: standard, featured (larger, shadow-lg), compact (no image). Uses radius-md and shadow-sm.

   **ISIBlock** (`isi-block.tsx`): Important Safety Information. Bordered container with "IMPORTANT SAFETY INFORMATION" heading in bold. Content area with smaller text. Optional expandable behavior (collapsed by default with "Read more" toggle). Yellow-amber left border to draw attention. This MUST be visually prominent — pharma regulatory requirement.

   **Disclaimer** (`disclaimer.tsx`): Small text block with subtle background. Type variants: general (gray bg), pharma (amber bg), legal (bordered). Font size body-sm.

   **CTA** (`cta.tsx`): Button/link component. Variants: primary (pfizer-blue-500 bg, white text), secondary (outlined), outline (transparent bg, blue border). Rounded, padded, hover states.

   **NavBar** (`nav-bar.tsx`): Horizontal bar with logo area (text placeholder), navigation links, and market indicator badge. Sticky top. White background with subtle bottom border.

   **Footer** (`footer.tsx`): Dark background (gray-900), light text. Sections: links grid, disclaimers area, copyright line, and MANDATORY adverse event reporting link (highlighted). The adverse event URL must be visually distinct (underlined, different color).

   **DataTable** (`data-table.tsx`): HTML table with proper thead/tbody. Caption above. Variants: standard, striped (alternating row colors), compact (tighter padding). Responsive: horizontal scroll on mobile.

   **ClaimReference** (`claim-reference.tsx`): Inline claim text with superscript footnote number linking to reference. Reference text appears below or in tooltip. Every claim MUST have a visible reference.

   **SectionHeader** (`section-header.tsx`): Title + optional subtitle. Alignment prop: left, center, right. Uses heading-lg for title, body-md for subtitle.

   **ContentBlock** (`content-block.tsx`): Rich text container with max-width constraint. Variants: standard, highlighted (light blue bg), callout (left border accent).

   **ImageBlock** (`image-block.tsx`): Image with required alt text, optional caption below, configurable width. Uses next/image if convenient, otherwise standard img with proper alt.
